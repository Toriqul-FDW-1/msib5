yang kemudian akan di upload kedalam github
menggunakan perintah :
git add -> untuk menambah data atau file
git commit -m -> untuk menambah tanda atau komentar
git push -u -> untuk memmastikan file masuk kedalam repo
git config --global user.name -> untuk memasukan settingan username kedalam git kita
git config -- global user.email -> untuk memmsaukan settingan email kedalam git kita
